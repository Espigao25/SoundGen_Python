SoundGen
========

SoundGen is a flexible burst mode ASK signal demodulator and frame processer, created with DVB-T in mind.

Look how easy it is to use:

    import project
    # Get your stuff done
    project.do_stuff()

Features
--------

-ASK demodulation
-Graph creation tools for debugging support
